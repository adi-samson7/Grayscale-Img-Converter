# 🖼️ Color Image to Grayscale Converter

A simple and interactive **Image Conversion Web App** built with **Streamlit**.  
This app allows you to upload a color image and instantly convert it to grayscale — perfect for quick edits or visual effects.

🚀 **Live Demo:**  
https://adi-samson7-grayscale-img-converter-gray-o1au13.streamlit.app/

---

## 📌 Features

- 📁 Upload color images or Use the camera
- ⚫ Convert images to grayscale instantly  
- 💾 Download grayscale result  
- 🧠 Simple, minimal, easy-to-use UI  

---

## 🧱 Built With

- **Streamlit** — UI and web deployment  
- **Python** — Backend logic  
- **Pillow (PIL)** — Image processing  

---

## 🛠️ How It Works

1. Upload a color image (jpg, png, etc.) or use the camera to click image
2. The app converts it to grayscale automatically  
3. Preview the converted image  
4. Download the grayscale image  

---

## 🧰 Requirements

- Python 3.7+  
- Streamlit  
- Pillow  

---

## 💻 Installation

To run the project locally:

```bash
git clone https://github.com/adi-samson7/Grayscale-Img-Converter.git
cd Grayscale-Img-Converter
pip install -r requirements.txt
streamlit run app.py
```

## ⚙️ Deployment

The app is deployed using Streamlit Community Cloud — just link your GitHub repo to deploy.
